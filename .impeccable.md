# Design Context — 吹牛骰子 (Chuiniu Dice)

## Target Audience
- 20-35岁年轻人，朋友聚会、KTV、酒吧场景
- 熟悉港式吹牛骰子规则的玩家
- 通过 Telegram Mini App 使用，主要移动端

## Use Cases
- 开房间邀请朋友一起玩骰子
- 摇骰子、叫点、开、判定输赢
- 输了接受随机惩罚（喝酒、真心话、大冒险）
- 聚会破冰、喝酒助兴

## Brand Personality & Tone
- **奢华中国风** — 红木、金箔、象牙骰子的质感
- **派对氛围** — 热闹、刺激、有竞争感
- **沉浸式游戏体验** — 像在真实的红木八仙桌上玩骰子
- **既传统又现代** — 中式美学 + 现代交互

## Aesthetic Direction
- **Tone**: Luxury Chinese gaming — 红与金的对比，深色背景如夜间雅座
- **Differentiation**: 不是卡通风，不是赛博朋克，而是有真实材质感的中式桌游体验。让人感觉像在高端私人会所玩骰子。
- **Key materials**: 红木纹理、金箔装饰、象牙/骨质骰子、宣纸质感
- **Typography**: Noto Serif SC（标题衬线感）+ 系统无衬线（正文可读性）

## Color Palette
- Background: #0F0F1A (深夜蓝黑)
- Surface: #1A1A2E (暗面板)
- Gold: #D4A853 (主色调 — 金)
- Red: #C41E2A (行动色 — 红)
- Dark Red: #8B1A1A (按下状态)
- Cream: #F5E6C8 (文本色 — 象牙白)
- Wood: #5C3A21 (木纹棕)

## Technical Constraints
- Vue 3 + Vite + TypeScript
- Tailwind CSS v4 (via @theme)
- Vant 4 mobile component library
- TG Mini App (320-430px width, touch-first)
- CDN images: images.walker-learn.xyz
